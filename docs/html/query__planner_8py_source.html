26  """Stages of computation compilation:
27 
28  There's the raw AlterTopo from the client. Call this the original plan.
29 
30  There's turning multi-node location specifiers (e.g. "on all nodes") into actual
31  node IDs. The client can do this. Call the result an expanded plan.
32 
33  This plan can be optimized. This involves replacing groups of operators with
34  equivalent groups, pushing operators back (towards sources) for partial aggregation,
35  converting logical operators into physical operators (sometimes adding operators),
36  and others. Call the result an optmized plan.
37 
38  There's placement. Call the result a placed plan. This can still include logical
39  operators that don't map exactly to physical operators.
40 
41  Last, there's putting in assorted plumbing. Call the result a concrete computation
42  plan. This must only have concrete physical operators. It will be cut up and sent
43  to the relevant dataplane nodes for execution according to above placement.
44 
45  Validation can happen at several points. Typechecking and suchlike can happen on the
46  expanded plan.
47  """
