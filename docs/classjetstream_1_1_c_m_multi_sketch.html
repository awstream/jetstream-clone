Detailed Description

exact level 0 has 2^(EXACT_LEVELS * BITS_PER_LEVEL) ints So if exact_levels =4, we have 2 ^ 10 bytes; for 5 exact levels, it'll be 4kb meanwhile each pane is depth * (width + 2) ints so if depth = 10 and width = 256, that's 10kb per pane
